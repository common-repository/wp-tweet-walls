== Description ==
Easily create and add beautiful customized Tweets and Twitter walls to your site in a few simple steps. 

= Features =
* Create up to 3 tweet walls to add to your site with 3 tweets each
* Fully customizable and allows changing the theme and colors used

= Premium Features =
* Create unlimited walls
* Create unlimited tweets
* View images, video and other media inside your tweets and tweet walls
* Create and add dynamic timelines to your site
* Create and add Twitter Tweet buttons, Follow buttons and Direct Message buttons

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress 'Plugins' > 'Add New' screen directly.
1. Activate the plugin through the 'Plugins' screen in your WordPress Dashboard
1. Navigate to the 'WP Tweet Walls option' in the sidebar to start creating your tweet walls

== Frequently Asked Questions ==
= How do I add a Tweet Wall to my site? =
Navigate to 'WP Tweet Walls' in the Dashboard menu after installing the plugin and click on the button to create a new Tweet Wall. Once created, copy the shortcode ([wptw_tweet_wall id="1"]) to your post or page and save the changes. That is all and your Tweet wall will now be displayed in the frontend.
